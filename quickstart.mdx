## 5-Minute Setup

<Steps>
  <Step title="Clone Repository">
    ```bash
    git clone https://github.com/yourusername/n-aible.git
    cd n-aible
    ```
  </Step>

  <Step title="Start Docker Services">
    ```bash
    docker-compose up -d
    ```
    
    This starts:
    - PostgreSQL on port 5432
    - Redis on port 6379
    
    <Check>Wait ~30 seconds for services to be ready</Check>
  </Step>

  <Step title="Setup Backend">
    ```bash
    cd backend
    
    # Create virtual environment
    python -m venv venv
    source venv/bin/activate  # On Windows: venv\Scripts\activate
    
    # Install dependencies
    pip install -r requirements.txt
    
    # Setup environment
    cp ../env_template.txt .env
    ```
    
    <Warning>
      Edit `.env` and add your API keys:
      - `OPENAI_API_KEY=sk-...`
      - `LLAMAPARSE_API_KEY=llx-...`
    </Warning>
    
    ```bash
    # Run database migrations
    cd database
    alembic upgrade head
    cd ..
    
    # Start backend server
    uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
    ```
  </Step>

  <Step title="Setup Frontend">
    Open a new terminal:
    
    ```bash
    cd frontend
    
    # Install dependencies
    npm install  # or pnpm install
    
    # Start dev server
    npm run dev  # or pnpm dev
    ```
  </Step>

  <Step title="Verify Installation">
    Open your browser and check:
    
    - **Frontend**: [http://localhost:3000](http://localhost:3000)
    - **Backend API**: [http://localhost:8000/docs](http://localhost:8000/docs)
    - **Health Check**: [http://localhost:8000/health](http://localhost:8000/health)
    
    <Check>You should see the login page!</Check>
  </Step>
</Steps>

## Alternative: Manual Database Setup

If you prefer not to use Docker:

<Tabs>
  <Tab title="PostgreSQL">
    ```bash
    # Install PostgreSQL (macOS)
    brew install postgresql
    
    # Install PostgreSQL (Ubuntu)
    sudo apt-get install postgresql
    
    # Create database
    createdb ai_agent_platform
    
    # Enable extensions
    psql ai_agent_platform -c "CREATE EXTENSION IF NOT EXISTS vector;"
    ```
    
    Update your `.env`:
    ```bash
    DATABASE_URL=postgresql://localhost:5432/ai_agent_platform
    ```
  </Tab>
  
  <Tab title="Redis">
    ```bash
    # Install Redis (macOS)
    brew install redis
    brew services start redis
    
    # Install Redis (Ubuntu)
    sudo apt-get install redis-server
    sudo systemctl start redis
    ```
    
    Update your `.env`:
    ```bash
    REDIS_URL=redis://localhost:6379
    ```
  </Tab>
</Tabs>

## Environment Configuration

<AccordionGroup>
  <Accordion title="Required Variables">
    ```bash .env
    # Database
    DATABASE_URL=postgresql://username:password@localhost:5432/ai_agent_platform
    
    # AI Services
    OPENAI_API_KEY=sk-...
    LLAMAPARSE_API_KEY=llx-...
    
    # Security
    SECRET_KEY=your-secret-key-at-least-32-characters
    
    # Environment
    ENVIRONMENT=development
    ```
  </Accordion>
  
  <Accordion title="Optional Variables">
    ```bash .env
    # Redis
    REDIS_URL=redis://localhost:6379
    
    # Google OAuth
    GOOGLE_CLIENT_ID=...
    GOOGLE_CLIENT_SECRET=...
    GOOGLE_REDIRECT_URI=http://localhost:3000/auth/google/callback
    
    # Additional AI
    ANTHROPIC_API_KEY=sk-...
    GEMINI_API_KEY=...
    
    # Storage (AWS/Wasabi)
    AWS_ACCESS_KEY_ID=...
    AWS_SECRET_ACCESS_KEY=...
    AWS_BUCKET_NAME=...
    ```
  </Accordion>
</AccordionGroup>

## Common Issues

<Warning>
  **"Could not connect to database"**
  
  - Check Docker is running: `docker-compose ps`
  - Verify DATABASE_URL in `.env`
  - Wait 30 seconds for PostgreSQL to initialize
</Warning>

<Warning>
  **"OpenAI API error"**
  
  - Check OPENAI_API_KEY in `.env`
  - Verify API key is valid
  - Check you have API credits
</Warning>

<Warning>
  **"Module not found" errors**
  
  - Activate virtual environment: `source venv/bin/activate`
  - Reinstall dependencies: `pip install -r requirements.txt`
</Warning>

<Warning>
  **Frontend won't start**
  
  - Clear npm cache: `npm cache clean --force`
  - Delete node_modules: `rm -rf node_modules`
  - Reinstall: `npm install`
</Warning>

## Development Workflow

Once everything is running:

```bash
# Terminal 1: Backend with auto-reload
cd backend
source venv/bin/activate
uvicorn app.main:app --reload

# Terminal 2: Frontend with hot reload
cd frontend
npm run dev

# Terminal 3: Watch logs
docker-compose logs -f postgres redis
```
