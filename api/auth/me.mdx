---
title: 'Get Current User'
description: 'Get information about the currently authenticated user'
---

Retrieve information about the currently authenticated user from the authentication cookie.

<Endpoint method="get" path="/api/auth/me" />

## Authentication

This endpoint requires authentication. The authentication cookie must be present in the request.

## Request Example

<RequestExample>
```bash cURL
curl -X GET 'http://localhost:8000/api/auth/me' \
  -b cookies.txt
```
</RequestExample>

<CodeGroup>
```javascript JavaScript
const response = await fetch('http://localhost:8000/api/auth/me', {
  credentials: 'include'  // Required for cookies
});

const user = await response.json();
```
```python Python
import requests

session = requests.Session()
# ... login first to get cookie ...
response = session.get('http://localhost:8000/api/auth/me')
user = response.json()
```
</CodeGroup>

## Response

<ResponseField name="id" type="integer" required>
Unique identifier for the user.
</ResponseField>

<ResponseField name="user_id" type="string | null">
Role-based user ID (e.g., `STUD-XXXXX`, `INSTR-XXXXX`, `ADMIN-XXXXX`).
</ResponseField>

<ResponseField name="email" type="string" required>
The user's email address.
</ResponseField>

<ResponseField name="full_name" type="string | null">
The user's full name.
</ResponseField>

<ResponseField name="username" type="string | null">
The user's username.
</ResponseField>

<ResponseField name="bio" type="string | null">
User biography.
</ResponseField>

<ResponseField name="avatar_url" type="string | null">
URL to user's avatar image.
</ResponseField>

<ResponseField name="role" type="string" required>
User role: `"student"`, `"professor"`, or `"admin"`.
</ResponseField>

<ResponseField name="is_active" type="boolean" required>
Whether the user account is active.
</ResponseField>

<ResponseField name="is_verified" type="boolean" required>
Whether the user's email is verified.
</ResponseField>

<ResponseField name="created_at" type="string" required>
ISO 8601 formatted timestamp of when the user account was created.
</ResponseField>

<ResponseField name="updated_at" type="string" required>
ISO 8601 formatted timestamp of when the user account was last updated.
</ResponseField>

## Use Cases

- Check if user is authenticated
- Get current user's role for role-based UI rendering
- Display user profile information
- Verify user account status

<Info>
This endpoint is useful for checking authentication status and retrieving user information without making a full profile request.
</Info>
